# Client

Next.js 14 frontend deployed on Vercel.

## Structure

- `src/app/` – Next.js App Router pages
- `src/components/` – React components
- `src/lib/` – Supabase client, utilities

## Dev

```bash
cd client
npm install
npm run dev
```

## Tech Stack

Next.js 14, Tailwind CSS, shadcn/ui, react-query v5, Supabase JS client.
